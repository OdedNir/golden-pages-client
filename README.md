# Golden Pages Website

## How to get started:

### Step 1:
Install Node.js on your computer.  
Run the command:
```bash
npm i -g npm
```
This will update the node package manager (npm).

### Step 2:
Clone the repository.  
Run the command:
```bash
npm install
```
Then:
```bash
npm start
```

### Finished!
Now the app will run on http://localhost:3000
